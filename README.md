# Description
This is a simple url shortener (duh) written in Javascript that gets the URL mappings from the `mapping.json` file
# Installing and running
To install the dependencies, do:
`npm install`
then run by using
`npm start`

# JSON mapping format
```json
{
  "shortUrl1" : "longUrl1",
  "shortUrl2" : "longUrl2"
}
```
